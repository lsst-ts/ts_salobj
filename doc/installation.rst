############
Installation
############

Install ts_salobj
=================

You may install ts_salobj using conda or pip::

    conda install -c lsstts ts-salobj[=version]

or::

    pip install ts-salobj[==version]

Configure Kafka
===============

Set environment variables ``LSST_KAFKA_BROKER_ADDR`` and ``LSST_SCHEMA_REGISTRY_URL``.
